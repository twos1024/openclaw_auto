# ClawDesk v0.1.3

## Highlights

- Setup Assistant now hands install-related flows directly into `Install Wizard` via deep link routing.
- Install and Overview entry points are unified, so install guidance no longer drops users into an unscoped page state.
- Dashboard diagnostics now include a backend-driven local endpoint probe, embed-phase status, external-open feedback, and platform-specific troubleshooting notes.
- Embedded Dashboard error handling is clearer across timeout, blocked iframe, unavailable gateway, and reachable endpoint states.

## Included Changes

- Added `probe_dashboard_endpoint` Tauri command for desktop-only dashboard reachability checks.
- Added a dedicated dashboard diagnostics model/service on the React side.
- Cleared stale external-open feedback when gateway status changes, preventing misleading diagnostics after restart/stop.
- Extended unit, integration, E2E, and Rust test coverage for setup handoff and dashboard probe behavior.
