# ClawDesk v0.2.0

## Highlights

- Added a first-class `Runbook` workspace that consolidates launch checks, blockers, recovery steps, and workflow context into one page.
- Added a global `Workspace Runtime Banner` so desktop runtime state, invoke bridge health, and critical blockers are visible across the app.
- Refactored `Setup Assistant` to reuse the same runbook model, reducing drift between guided actions and page-level workflow state.
- Integrated Install, Logs, and Settings with shared workflow context so users can see blockers and next actions without switching mental models.

## Included Changes

- Added `runbookService`, `useRunbook`, and shared workspace types for reusable launch-check and blocker modeling.
- Added reusable UI primitives for page hero, notice banners, status badges, and surface cards to normalize interaction patterns.
- Added `RunbookPage` plus contextual runbook panels inside Install, Logs, and Settings.
- Improved runtime diagnostics surfacing in Settings and shell-level runtime messaging in AppShell.
