# ClawDesk v0.3.0

## Highlights

- Aligned the workspace guidance model with the current SPEC so setup, runbook, and dashboard recommendations stay consistent.
- Added a visible recommended next action to dashboard diagnostics, including a clearer path when Gateway is not running.
- Embedded the Setup Assistant launch checks directly in the dialog and tightened modal keyboard accessibility.
- Expanded runtime banner and Settings diagnostics to show bridge source and readiness in friendlier language.

## Included Changes

- Added `DashboardRecommendedAction` to the dashboard diagnostics model and surfaced it in the Dashboard diagnostics panel.
- Added `RunbookLaunchChecks` to the Setup Assistant dialog so launch checks are visible without leaving the modal.
- Improved runtime banner, Settings, and runbook metadata to show bridge source, invoke bridge, and Tauri shell state.
- Added focus trapping and Escape-to-close support to the shared `ModalDialog`.

## Verification
