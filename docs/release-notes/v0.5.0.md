# ClawDesk v0.5.0

## Highlights

- Synced the app version to `0.5.0` across the frontend, Tauri config, Cargo manifest, and lockfiles.
- Hardened the key workspace hooks with `try/catch/finally` so unexpected load/save failures do not leave the UI stuck in a loading state.
- Fixed settings normalization so `preferredInstallSource` is preserved from backend data instead of being overwritten during load.

## Stability

- `useRunbook()` now clears loading state even if overview loading throws unexpectedly.
- `useSettingsForm()`, `useConfigForm()`, `useAppSettingsSnapshot()`, `useEnvironmentSnapshot()`, and `useInstallFlow()` now recover cleanly from unexpected async failures.
- `useInstallFlow()` now synthesizes a readable error result if install execution throws before the Rust service returns a structured response.
