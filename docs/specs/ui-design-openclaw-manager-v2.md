## 1. 设计语言分析（v1.0.70 现状）

| 维度 | 现状描述 |

## 5. 交互规范

### 5.1 导航交互

| 触发 | 行为 | 视觉反馈 |
|------|------|----------|
| 点击侧边栏菜单项 | 切换到对应页面 | 左侧 3px 橙色竖条 + 文字变橙 |
| 点击 Logo 区域 | 返回首页 | - |
| 点击底部统计卡片 | 跳转实例列表 | 卡片 hover 变灰底 |

### 5.2 对话框交互

| 触发 | 行为 | 动画 |
|------|------|------|
| 点击「+ 新建实例」 | 打开创建向导 Modal | fade-in 200ms + slide-up |
| 点击步骤编号 | 跳转到对应步骤 | slide-left/right |
| 点击「下一步」 | 校验当前步 → 进入下一步 | slide-left |
| 点击 ✕ 或遮罩 | 关闭对话框（需二次确认如有内容） | fade-out |

### 5.3 AI 对话交互

| 触发 | 行为 | 视觉反馈 |
|------|------|----------|
| 发送消息 | 消息气泡出现 + 流式回复 | 打字机效果 |
| 切换模型 | 下拉菜单选择 | 顶部模型名称更新 |
| 切换 Tab | LLM ↔ Agent | Tab 下划线滑动 |
| 新对话 | 清空当前会话 | 右侧内容区淡出淡入 |

### 5.4 语言切换

```
点击「简体中文 ▼」→ 弹出下拉菜单:

  ┌──────────────┐
  │ 🇨🇳 简体中文  ✓ │  ← 当前选中 (橙色文字)
  │ 🇺🇸 English    │
  │ 🇯🇵 日本語     │
  └──────────────┘

切换后: 全局 UI 文字即时刷新，无需重启
```

## 8. v2.0 UI 重构建议

| 编号 | 重构项 | 现状问题 | 建议方案 |
|------|--------|----------|----------|
| D1 | 样式系统 | 全量 inline styles，不可维护 | 迁移 Tailwind CSS，提取 Design Tokens |
| D2 | 组件库 | 无统一组件库 | 基于 shadcn/ui 或 Radix 构建 |
| D3 | 空状态 | 只有文字提示 | 增加插图 + 引导按钮 + 动画 |
| D4 | 实例卡片 | 未实现（空状态） | 设计卡片组件：状态灯、模型标签、操作菜单 |
| D5 | Token 图表 | 仅文字「暂无数据」 | 接入 Recharts，绘制用量趋势折线图 |
| D6 | 对话体验 | 基本功能 | 增加 Markdown 渲染、代码高亮、文件预览 |
| D7 | 深色模式 | 不支持 | 基于 CSS variables 实现 light/dark 主题 |
| D8 | 动效 | 几乎无动效 | 页面切换 transition、按钮 hover 效果 |
| D9 | 图标系统 | 混用 Emoji + Unicode | 统一使用 Lucide Icons |
| D10 | 侧边栏切换 | 对话页独立侧边栏布局 | 统一侧边栏，对话页增加二级面板 |
